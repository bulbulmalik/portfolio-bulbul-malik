RAG Knowledge Base Demo

This project demonstrates a practical retrieval-augmented generation system. It retrieves the most relevant knowledge documents using semantic search and then generates concise answers from the retrieved context.

It is built with:

FastAPI SentenceTransformers google/flan-t5-small RAG

Run the app locally to query the demo API or inspect the source in the project folder.

Open app.py

How it works

1. Load markdown knowledge documents.

2. Encode documents with semantic embeddings.

3. Retrieve top matching passages for each user query.

4. Generate an answer from the retrieved context using a text generation model.
